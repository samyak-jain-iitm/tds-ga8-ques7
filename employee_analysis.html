1. Analysis Logic (Python Code)

The following Python code was used to load the dataset, filter for the R&D department, and generate the distribution visualization.

import pandas as pd
import matplotlib.pyplot as plt

# Load Data
df = pd.read_csv('employee_data.csv')

# Calculate R&D Count
rd_count = df[df['department'] == 'R&D'].shape[0]
print(f"R&D Department Count: {rd_count}")

# Create Histogram
plt.figure(figsize=(10, 6))
df['department'].value_counts().plot(kind='bar', color='#4c72b0')
plt.title('Department Distribution')
plt.xlabel('Department')
plt.ylabel('Count')
plt.show()
